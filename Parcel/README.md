# Modulo 3 - Bundling - Parcel

Esta es una aplicacion simple para la entrega del modulo 3

## Instrucciones

-   `npm install`

#### Dev-Server

-   `npm run start`

#### Build

-   `npm run build:dev` o `npm run build:prod`
-   Ir a http://localhost:1234

#### Bundle Analyzer

-   `npm run build:prod`
-   Abrir el fichero `dist/report.html`
